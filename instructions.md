# Instructions

How to get started with Visual Studio Code for Windows

Open any search engine and look for vscode or type in this url: https://code.visualstudio.com/

Click download and follow instructions

<img width="2559" height="1343" alt="image" src="https://github.com/user-attachments/assets/2a6ec1ed-3425-48ff-b9d3-934b55c8f3bc" />

Once downloaded you'd want to create a folder, this folder is where you'll store your future projects

After your folder is created click OPEN FOLDER and look for said folder

<img width="2559" height="1392" alt="image" src="https://github.com/user-attachments/assets/c1d650c7-350c-49f6-a63d-593f7266ff7c" />

On the left screen side you'd be able to access your projects and files, all you got to do now is right click and select "New File..."

Name this new file whatever you'd like and end it with the proper file end, in this case I'll use .c since I'm working with C program

Depending on the language you'd want to download the proper language extensions and Code Runner through VSC which is also found on the left side

<img width="2601" height="1370" alt="image" src="https://github.com/user-attachments/assets/d90c74bc-7b59-432b-a3b5-7436bdd67f5d" />

Happy coding!
